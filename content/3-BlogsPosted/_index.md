During my participation in the FCAJ program, I completed and published three technical blog posts on the AWS Study Group community to share knowledge and practical experience:

### [Blog 1 - Automating Recommendation Engine Training with Amazon Personalize and AWS Glue](3.1-Blog1/)
This blog introduces a serverless architecture to automate data ingestion and training for a recommendation system. It details using AWS Glue to crawl, clean, and consolidate data from different sources into S3, and then using Amazon Personalize to automatically train recommendation models.

### [Blog 2 - Session Policies in Amazon EKS Pod Identity](3.2-Blog2/)
This blog discusses the session policies feature in Amazon EKS Pod Identity. It explains how to dynamically narrow IAM permissions for individual pods using a single shared IAM role, making it easier to apply the least privilege principle in large Kubernetes environments.

### [Blog 3 - Optimizing LLM Inference on Amazon SageMaker AI with BentoML LLM Optimizer](3.3-Blog3/)
This blog details a workflow to optimize deployment costs and inference performance for open-source Large Language Models (LLMs). It demonstrates utilizing the BentoML LLM Optimizer on Amazon SageMaker AI to run automated load-testing and find the best AWS instance type and quantization configurations.
