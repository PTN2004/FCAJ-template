### Week 6 Objectives:

* Understand serverless object storage systems using Amazon Simple Storage Service (S3).
* Deploy and configure security bounds using S3 Bucket Policies and Block Public Access controls.
* Deploy, publish, and secure a static website hosted entirely inside an S3 bucket.

### Week 6 Achievements:

* Created and configured globally unique Amazon S3 storage buckets inside target AWS regions.
* Mastered granular security controls of S3, managing S3 Block Public Access variables and understanding policy scopes.
* Configured serverless static website hosting on S3, hosting functional HTML/CSS pages without provisioning a web server.
* Deployed JSON Bucket Policies to allow secure, public read-only access to objects while protecting bucket administration scopes.
